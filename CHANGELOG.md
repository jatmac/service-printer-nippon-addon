# Changelog

All notable changes to this project will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [1.1.2] - 2025-12-05

### Added

- Nippon printer specification documentation
- Mileage usage tracking tests for printer status monitoring
- Detailed status mapping documentation for error handling

### Changed

- Reorganized documents inside doc folder for better structure
- Enhanced printer status reporting with comprehensive status information
- Improved printer status handling with detailed error states

## [1.1.1] - 2025-11-27

### Fixed

- Updated package name to `neo-printer-nippon-addon` for consistency
- Replaced Unicode icons with ASCII equivalents for better terminal compatibility

### Removed

- Setup script for Nippon Printer addon (no longer needed)

## [1.0.0] - 2025-11-27

### Features

- Initial release of Nippon Thermal Printer Node.js addon
- Native addon implementation using NPrinterLib
- Support for thermal receipt printing
- Printer enumeration and device discovery
- Status monitoring and error reporting
- Image printing capabilities
- Receipt formatting utilities
